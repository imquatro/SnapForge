# SnapForge

Browser and desktop capture workflow app.

## What SnapForge Does

SnapForge is a Windows desktop capture tool focused on fast screenshot workflows.

### Main Features

- Global hotkeys for capture and overlay actions.
- Quick capture overlay with recent screenshots.
- Recent gallery with selection, zoom, open, edit, send, and delete actions.
- Built-in image viewer with zoom, pan, and keyboard navigation.
- Built-in image editor with brush drawing, color picker, undo, save, and crop (cut).
- Capture notification toast (preview + sound).
- Tray mode support (start minimized, open from tray).
- Built-in update check using GitHub Releases.
- Installer-based deployment with uninstaller support.
